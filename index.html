Sublime ColorPicker

A color picker plugin for Sublime Text 2

View the Project on GitHub weslly/ColorPicker

Installation

Install this repository via Package Control

Usage

To insert or change a selected color, use:

  • Linux: ctrl+shift+c
  • Windows: ctrl+shift+c
  • OS X: super+shift+c
